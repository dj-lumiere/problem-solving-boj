### 문제 설명

<p>유명한 가지 요리사 민규는 오늘도 요리에 사용할 볼록다각형 모양 가지를 손질하고 있다. 민규는 완벽하게 손질된 가지를 만들기 위해 다음 과정에 따라 가지를 자른다.</p>

<ul>
	<li>가지를 이루는 각 변의 중점을 이어 새로운 볼록다각형을 만든다. 항상 가지를 자르는 것이 가능함을 보일 수 있다.</li>
	<li>만들어진 볼록다각형에 포함되지 않는 부분을 잘라 버린다.</li>
</ul>
